What is CSS?

CSS (Cascading Style Sheets) is a language used to style HTML documents. It allows you to control the layout, colors, fonts, and much more, making your web pages visually appealing.

CSS Selectors

Selectors are used to target HTML elements you want to style. Some common selectors include:

  • element: Targets a specific HTML element, e.g., p
  • .class: Targets elements with a specific class, e.g., .highlight
  • #id: Targets elements with a specific ID, e.g., #main

CSS Box Model

The box model describes how the size of elements is calculated. It consists of:

  • Content: The actual content of the element
  • Padding: Space between content and the border
  • Border: The edge of the element
  • Margin: Space outside the border
